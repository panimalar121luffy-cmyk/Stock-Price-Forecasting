DESCRIPTION:

During my internship, I developed a Python‑based Stock Price Forecasting program that applies machine learning techniques to predict future stock trends using historical data. The project uses libraries such as pandas and NumPy for data handling, yfinance for fetching real‑time stock market data, scikit‑learn for preprocessing, and TensorFlow/Keras for building an LSTM neural network tailored for time‑series forecasting. The workflow includes data normalization, sequence generation, model training, and evaluation, followed by visualizations of actual versus predicted prices using matplotlib. To keep the program modular and internship‑ready, I structured the code into clear steps: data loading, preprocessing, model building, training, and result visualization. This project strengthened my skills in time‑series analysis, deep learning, and financial data modeling, while demonstrating how AI can be applied to real‑world domains like stock market prediction. It aligns with industry practices in quantitative finance and showcases the ability to combine data science with business intelligence for actionable insights.
